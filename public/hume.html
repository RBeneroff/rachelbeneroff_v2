HUME

MOBILE APP PROPOSAL SUMMARY

PROJECT OVERVIEW

Hume requires a new mobile application to replace their current Arketa solution. They seek greater creative control, better content management, and an app that truly represents their premium brand identity while maintaining operational efficiency.

SOLUTION OPTIONS

CUSTOM DEVELOPMENT

Investment: $65,000
Timeline: 27 weeks
Monthly Costs: $200-500

Key Features:

  • Complete creative control over design
  • Custom-built content management system
  • Seamless third-party integration
  • Subtle visual effects and responsive interface elements
  • Comprehensive data migration from Arketa
  • 90-day post-launch support

Ideal For: Clients prioritizing distinctive brand expression, unique functionality, and long-term flexibility without platform constraints.

ENHANCED PLATFORM

Investment: $42,000
Timeline: 15 weeks
Monthly Costs: $300-800

Key Features:

  • Significantly more customization than Arketa
  • Established infrastructure with proven reliability
  • Easier content management through platform tools
  • Regular platform updates and improvements
  • Smooth data migration from Arketa
  • 60-day post-launch support

Ideal For: Clients seeking a balance between customization and operational efficiency, with faster implementation and lower initial investment.

PLATFORM RECOMMENDATIONS

  • Primary: Walla - Modern design, strong customization options, built for boutique wellness
  • Secondary: Kenko - User-friendly with good customization capabilities
  • Tertiary: Glofox or WellnessLiving - Comprehensive features for multi-location management

CUSTOM THIRD-PARTY OPTIONS

Recommended third-party scheduling systems for integration with custom development:

  • Primary: Zenoti - Comprehensive API capabilities, enterprise-level solution
  • Secondary: WellnessLiving - Robust API access with strong support for Arketa migration
  • Tertiary: Mindbody or Book4Time - Established solutions with extensive API documentation
